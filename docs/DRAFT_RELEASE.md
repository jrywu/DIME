
## 修正與改進 (v1.3.588 之後)

### 新功能

- **全新 Windows 11 風格設定介面**：以側邊欄 + 卡片式佈局取代舊版 PropertySheet 對話方塊，支援深色模式、雙層導覽（分類卡片→子頁面）、候選字預覽、色彩選擇器及響應式側邊欄折疊（詳見 [docs/NEW_DIME_SETTINGS.md](docs/NEW_DIME_SETTINGS.md)）
- **全半形及繁簡轉換提示常駐顯示**：全半形切換與繁簡轉換的桌面提示不再受「顯示桌面提示」選項限制，因為這兩項狀態切換無系統語言列圖示，桌面提示是唯一的視覺回饋
- **行列字根表更新至 v2026-1.02**：新增 CJK Radicals Supplement（⺀–⺿）hg0–hg9 跳脫輸入、擴充 H/I/J 區 +9111 字、622 筆按鍵修正；Ext-EFG 更名為 Ext-E~J 以反映擴充範圍

### 修正

- **修正非可編輯視窗顯示中英切換提示**：對話方塊、訊息方塊等無輸入欄位的視窗取得焦點時，不再觸發中英模式提示氣泡
